# App Store Screenshots Guide

## Required Sizes

### iOS
- **6.7" Display** (iPhone 15 Pro Max): 1290 x 2796 px
- **6.5" Display** (iPhone 14 Plus): 1284 x 2778 px
- **5.5" Display** (iPhone 8 Plus): 1242 x 2208 px
- **12.9" iPad Pro**: 2048 x 2732 px (if supporting iPad)

### Android
- **Phone**: 1080 x 1920 px minimum
- **Feature Graphic**: 1024 x 500 px
- **TV Banner** (if applicable): 1280 x 720 px

## Design Guidelines

### Colors
- Background: Dark (#0f0f0f to #1a1a2e gradient)
- Silver accent: #94a3b8
- Gold accent: #fbbf24
- Success/Positive: #22c55e
- Text overlays: White with subtle shadow

### Typography
- Headlines: Bold, 48-64px
- Subtext: Regular, 24-32px
- Use SF Pro Display or similar clean sans-serif

### Layout
- Keep 100px safe zone at top/bottom for device frames
- Center text overlays
- Use device mockups for professional appearance
- Consistent positioning across all screenshots

### Do's
- ✅ Use realistic sample data (not "Test" or "Lorem ipsum")
- ✅ Show meaningful totals (e.g., 127.5 oz silver, $4,235 value)
- ✅ Use actual product names (American Silver Eagle, etc.)
- ✅ Keep interface clearly readable
- ✅ Highlight the key feature in each screenshot

### Don'ts
- ❌ Show empty states
- ❌ Use placeholder text
- ❌ Crop important UI elements
- ❌ Use inconsistent styling between screenshots
- ❌ Include any real personal data
